Support

The following is a link to FL Studio's support page. Come here if you encounter any technical problems with the software.
As shown on the site, they recommend you follow a few steps before contacting support.

  1. Check Frequently Asked Questions
  2. Create a Support Ticket for account/purchase help
  3. Create a message board post for technical issues
  4. If none of the above apply, you can create a generalized support ticket where you can select what your issue is from a drop down menu.
